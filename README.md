En este proyecto la idea fue crear una herramienta de verdad utilizando las bases logicas de lo que estaba aprendiendo, poder definir funciones que evaluen precios y devuelvan un resultado, que se calcule un promedio, que se identifique objetos y valores del mas caro y el mas barato. Al final una ultima función final que llama a todas las funciones aisladas y lo resuelve todo en un reporte simple y legible. Para ejecutarlo usamos python analizador.py.
